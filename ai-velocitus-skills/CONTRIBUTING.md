# Contributing to AI Velocitus Skills

Thank you for your interest in contributing. This repository is maintained by **SK / AI Velocitus** and contributions are welcome from Indian finance and compliance professionals.

---

## Ways to Contribute

### 1. Report an Issue
Found an error in a regulatory citation? A missing obligation? A penalty figure that's been updated by MCA/SEBI?

Open a **GitHub Issue** with:
- The specific section/regulation that needs updating
- The correct current position (with source — MCA circular, SEBI notification, etc.)
- Date of the change

### 2. Suggest a New Skill
Have a domain that deserves its own skill? Open an Issue with:
- Proposed skill name and domain
- Key regulatory frameworks it should cover
- Target audience (CA firms, listed companies, NBFCs, etc.)

### 3. Improve an Existing Skill
Submit a **Pull Request** with your changes to `SKILL.md` or reference files. Please:
- Describe what changed and why
- Cite the regulatory source for any factual updates
- Do not change the author attribution in the SKILL.md footer

### 4. Add a Reference File
If you want to contribute a new reference document (e.g., a penalty schedule, a form index), open a PR adding it to the relevant `references/` folder and update the SKILL.md to reference it.

---

## Code of Conduct

- Contributions must be factually grounded — cite your sources
- Do not submit AI-hallucinated regulatory content
- Respect the author attribution on all files
- Commercial forks must retain the MIT License and copyright notice
