## How to Install a Skill

1. Navigate to the skill folder and download the `.skill` file
2. Open **Claude.ai** → **Settings** → **Claude's Skills** → **Install Skill**
3. Upload the `.skill` file
4. That's it — Claude will trigger the skill automatically when relevant

> Skills work with Claude.ai Pro, Team, and Enterprise plans.
